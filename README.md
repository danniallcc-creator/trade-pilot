# 外贸领航者 · Trade Pilot

面向中国外贸从业者的分级知识库平台。为小白、新手、高手三类用户提供体系化的成长内容、实操工具、培训课程与真实案例。

## 在线预览

https://danniallcc-creator.github.io/trade-pilot/

## 站点结构

- `index.html` — 首页（三层能力 / 课程体系 / 工具库 / 专家圆桌）
- `level1.html` — Level 01 · 小白支持站（入行 0–6 个月）
- `level2.html` — Level 02 · 新手成长营（6 个月–2 年）
- `level3.html` — Level 03 · 高手跃迁塔（2 年以上 / 团队 / 创业）
- `_shared.js` — 分层页共享数据（TIER_DATA）与 Tailwind 配置

## 技术栈

- React 18 + Babel Standalone（浏览器内运行，无需构建）
- Tailwind CDN 3.4（黑金主题定制）
- Google Fonts：Inter + Noto Serif SC
- 纯静态部署（GitHub Pages）

## 本地预览

```bash
cd trade-pilot
python3 -m http.server 8000
# 访问 http://localhost:8000/
```

## 内容体系

三层能力模型，共 156 课时、216 个实操知识点、20+ 工具平台。
